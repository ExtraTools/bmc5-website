# BMC5 Website

Сайт Minecraft сервера BMC5.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub: `bmc5-website`
2. Загрузи файлы в репозиторий
3. Включи GitHub Pages в настройках репозитория

## Файлы

- `index.html` - главная страница
- `index.css` - стили
- `app.js` - логика (статус сервера)
